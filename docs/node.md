# Node.js

## Install Node.js

```bash
bash -c "$(curl --location https://docs.impossible98.win/node/install.sh)"
```

## Configure

```bash
yarn config set registry http://mirrors.cloud.tencent.com/npm/
```

## Uninstall Node.js

```bash
bash -c "$(curl --location https://docs.impossible98.win/node/uninstall.sh)"
```
